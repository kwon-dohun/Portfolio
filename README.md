# Dohun's Portfolio Website

This project is a personal portfolio website built for the Web Programming course.

It introduces who I am, the projects I have worked on, and the technologies I am currently learning.

## Features

- Clean and responsive layout
- Project showcase section
- Skills overview
- Contact page with basic information

## Tech Stack

- HTML5
- CSS3

## Structure

- `index.html` – main landing page
- `projects.html` – project showcase
- `skills.html` – skills and tools
- `contact.html` – contact information
- `styles.css` – styling
- `photo.jpg` – profile image

## Notes

This website focuses on building a clear structure using HTML and CSS.
